## How to Use

1. Copy the entire prompt block (title through tags)
2. Go to [suno.ai](https://suno.ai)
3. Click "Create" and paste
4. Set duration to match (e.g., 90 seconds)
5. Generate and download
6. Register with: `node tools/track_register.mjs public/music/<filename>.mp3 --project <id> --template <name> --bpm <num>`
